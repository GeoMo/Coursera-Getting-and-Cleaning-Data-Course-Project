# Course Project Code Book
## Original Data
There original data represent data collected from the accelerometers from the Samsung Galaxy S smartphone .  For detailed description of the original dataset, please see features_info.txt in the zipped dataset file. The 561 Features are normalized and bounded within [-1,1.
## Raw data
The training and test subsets of the original dataset were combined to produce merged raw dataset. Selecting from merged raw dataset only the columns representing means and  standard deviations,  subject identifiers subject  and activity labels a  processed raw data with 68 variables is obtained.
## Tidy data
Tidy data set contains the average (numeric value) of all feature (mean, standard deviation values) of the raw data for each subject and each activity. All the values are means, aggregated over 30 subjects and 6 activities, thus the resulting dataset is 180 rows by 68 columns.
Note that this data is in the wide format, respecting the rules of a tidy data, as is presented in lectures (Hadley Wickham's Tidy Data paper) and discussed in to the Course Forum on subject “Tidy data and the Assignment” https://class.coursera.org/getdata-013/forum/thread?thread_id=31.
Also the variable names respect the same principles (lower case).
